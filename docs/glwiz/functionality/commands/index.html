Provides utilities for executing system commands, including sudo and user-level operations.

This submodule contains functions to run commands with elevated privileges or as the current user, handling tasks like copying files or executing shell scripts. It is used across other submodules for operations requiring system-level access.

use glwiz::functionality::commands::run_sudo_command;
