The core module for the GNU/Linux Config Wizard, containing utilities for system configuration.

This module organizes functionality into submodules for tasks such as software installation, shell configuration, iptables setup, and ZRAM swap management. Each submodule provides specific tools to automate post-installation setup for GNU/Linux systems, ensuring a streamlined and user-friendly experience.
