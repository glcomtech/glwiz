pub fn read_input() -> String
Expand description

Reads a line of input from standard input (stdin).

This function captures a single line of user input in the “gnulinwiz” project, used for interactive tasks like prompting for software lists or overwrite confirmations. It flushes stdout to ensure prompts are displayed and expects input to be valid UTF-8, panicking on I/O errors for simplicity.

§Returns

A String containing the user’s input, including the trailing newline.

§Panics

Panics if reading from stdin fails (e.g., due to I/O errors). This is intentional for simplicity, as stdin is expected to be available in an interactive context.

§Example

use gnulinwiz::functionality::prog_fun::read_input;
let input = read_input();
println!("User entered: {}", input.trim());
